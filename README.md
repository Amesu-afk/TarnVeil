## ✨ Возможности

- 💬 Личные сообщения и групповые сообщества
- 🎧 Голосовые и видеозвонки
- 🧵 Обсуждения и темы внутри сообществ
- 📎 Вложения: картинки и файлы
- 🔔 Push-уведомления
- 🛡️ Блокировка и жалобы, удаление аккаунта

## 📲 Установка (Android)

1. Нажми **[Скачать для Android](https://amesu-afk.github.io/TarnVeil/)** и открой `.apk`.
2. Разреши установку из этого источника, если телефон спросит.
3. Play Protect может предупредить → **«Подробнее» → «Всё равно установить»** (это нормально для приложений вне Google Play).
4. Открой TarnVeil, создай аккаунт — готово 👻

> Минимум Android 7.0. Размер ~5 МБ.
